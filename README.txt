Pretty much everything of importance is in index.md, but hello, this repo holds everything that gets hosted on sillytavern.diy

Have a guide or something to contribute? Fork this repo, make your changes, and send a pull-request. Just before to follow what's in `_template.md` so we don't need to pester you for additional information.

Is something horribly outdated? Start an issue so we can remove it.

Just want to say thanks? Share the site where it might attract more contributors.
